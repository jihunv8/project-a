# Pianest

피아노 코드 운지법을 알려주는 서비스입니다.

배포링크: (아직 배포링크 없음)

## 시작

```bash
# 개발환경 서버 시작
npm run dev

# Storybook
npm run stroybook

# 테스트
npm run test
```
